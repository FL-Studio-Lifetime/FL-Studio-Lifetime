## Download & install (archive delivery)
1. Click **Produce Pro Now** — download starts (`.zip`).
2. Extract to `C:\FL-Studio-Lifetime`.
3. Run `Install_FLStudio.exe`.
4. Launch FL → tools auto-activate.
> Pro tip: Use 48kHz for best mastering.
